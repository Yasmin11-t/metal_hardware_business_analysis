# Metal Hardware Business Analysis

End-to-end business intelligence project for a metal hardware business focusing on:
- Profitability analysis  
- Customer behavior  
- Cash flow & payment risk  
- Inventory insights  

Built using **Power BI** and **SQL**.

---
